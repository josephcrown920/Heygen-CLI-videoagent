**Routes and their files:**
- `/` Dashboard, `/videos` VideoLibrary, `/create` CreateVideo, `/create/prompt` CreateFromPrompt
- `/models` ModelHub, `/generate/:modelId` GenerationStudio, `/creations` Creations
- `/canvas` Canvas (drag/resize/export PNG, layers, text/rect/image)
- `/scenes` ScenesProps (27 built-in presets: environments, urban, outfits, lighting, props)
- `/references` References (upload images → localStorage as base64, tag/search)
- `/lyrics` Lyrics (line editor, timestamps, SRT export, HeyGen script copy)
- `/live-avatar` LiveAvatar (WebRTC docs, migration checklist, code examples)
- `/cli` CLI (code generator for fal.ai/HeyGen/Modal/RunPod)

**Models in registry:** Kling 2.1 Master/Standard, Kling 1.6 Pro/Standard/Nano, LTX-Video, LTX-Video Turbo, Seedance 1.0, Wan 2.1 1.3B/14B, Hunyuan, MiniMax, Pixverse, CogVideoX, Luma Ray 2 Flash (video); Seedream 3.0, FLUX 1.1 Pro/Schnell/Dev, SD 3.5 Large, Ideogram v3, Recraft v3, Imagen 4, Kolors, AuraFlow (image)
